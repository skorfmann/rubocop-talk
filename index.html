What is Rubocop?

RuboCop is a Ruby static code analyzer.

Why should I care?


class MyClass
  attr_accessor :name

  def initalize name
    self.name = name
  end

  def name
    name.upcase
  rescue
  end

  def greet
    return 'Hello'+name
  end
end

Why Should I care?

Community Stylguides

Ruby Example: 


# starting point (line is too long)
def send_mail(source)
  Mailer.deliver(to: 'bob@example.com', from: 'us@example.com', subject: 'Important message', body: source.text)
end

# good (normal indent)
def send_mail(source)
  Mailer.deliver(
    to: 'bob@example.com',
    from: 'us@example.com',
    subject: 'Important message',
    body: source.text
  )
end
